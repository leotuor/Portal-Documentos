# UCEFF API

  * Primeiramente Intalar com o comando:
    ```
    npm i
    ```
  * Modelo para o arquivo **.env** no diretorio principal:
  
    ```
    POSTGRES_HOST=
    POSTGRES_DB=
    POSTGRES_USERNAME=postgres
    POSTGRES_PASSWORD=
    POSTGRES_PORT=5432
    API_PORT=3333
    API_HOST=http://localhost
    ```
